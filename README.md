# Attendance Tracking System
*Collaborative Major Subject Project | Completed during my 3rd Year 1st sem*

This is a web-based attendance tracking application developed as part of a multi-semester collaboration guided by our major subject instructor, Prof Lucman Abdulrachman. The system is built to provide teachers with an efficient, digital dashboard to track student presence, view statistics, and manage classroom data.

## Project Structure & Architecture
* **Frontend Design:** Built using the Quasar Framework and Vue.js for a modern, responsive user interface.
* **Type Safety:** Utilizes TypeScript to ensure clean code architecture and reliable component scripts.
* **Backend Database:** Integrated with Firebase for secure data hosting, authentication, and live tracking updates.

## My Contributions
* **Component Development:** Collaborative creation of interactive tab panels and interface views.
* **Data Visualizations:** Worked with tracking data to prepare structured logic for student averages and attendance summaries.
* **System Integration:** Assisted in organizing scripts and integrating local components inside VS Code.
